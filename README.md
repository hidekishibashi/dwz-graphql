# DWZ GraphQL on Rails

## Versions

| name  | version |
|:------|:-------:|
| Ruby  | 3.1.2   |
| Rails | 7.0.4   | 
| mysql | 8.0.21  |

## How to Start

#### Build Docker

```
docker compose build
```

#### Create Database

```
docker compose rub web rails db:create
```

#### Start Rails Application

```
docker compose up
```
